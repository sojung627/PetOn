# PetOn (반려동물 종합 플랫폼)

## 📌 프로젝트 소개
반려동물 관련 정보 제공 및 커뮤니티 기능을 제공하는 웹 서비스입니다.

## 🛠 기술 스택
- Backend: Java, Spring Boot, MyBatis
- Frontend: JSP, HTML/CSS, JavaScript
- DB: PostgreSQL
- 배포: Railway

## ✨ 주요 기능
- 회원가입 / 로그인 / 회원정보 수정
- 게시판 / 댓글 / 답글 기능
- 반려동물 등록 및 관리

## ⚠️ 개선 필요 사항
- 결제 API 미구현
- 일부 UI/UX 개선 필요
- 시간 포맷 및 댓글 줄바꿈 개선 예정

## 💡 느낀 점
초기 프로젝트로 구조적인 한계를 경험했고,  
현재 리팩토링을 통해 코드 개선 및 기능 확장을 진행 중입니다.
